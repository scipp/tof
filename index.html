Tof#

A simple tool to create time-of-flight chopper cascade diagrams

✨ Try it in your browser ✨#

Installation#

To install Tof and all of its dependencies, use

pip install tof

conda install -c conda-forge -c scipp tof

Get in touch#

  • If you have questions that are not answered by these documentation pages, ask on discussions. Please include a self-contained reproducible example if possible.
