:transparent:`Tof`
******************

A simple tool to create time-of-flight chopper cascade diagrams.

Installation
============

You can install from ``pip`` using

.. code-block:: sh

   pip install tof

You can install from ``conda`` using

.. code-block:: sh

   conda install -c conda-forge -c scipp tof
